üí° Tip:

üí° Quick Add Examples:

üõçÔ∏è Orders Management

Checkout & Create Order

POST /api/orders — Auth: JWT
Create a new order from cart items (checkout process)

üìç Delivery Address

Enter your M-Pesa registered phone number

üìã Checkout Process:

  1. Add items to cart using "Add to Cart"
  2. Review cart using "Get Cart"
  3. Update quantities if needed
  4. Choose service type and payment method
  5. Fill delivery address if choosing delivery
  6. Click "Checkout & Create Order" to complete purchase
  7. Cart will be automatically cleared after successful order

View My Orders

GET /api/orders/user — Auth: JWT
Get all orders for the logged-in user

üí° Order Status Guide:

  • pending - Order received, waiting for confirmation
  • confirmed - Order confirmed by restaurant
  • preparing - Food is being prepared
  • out-for-delivery - Order is on the way
  • delivered - Order completed successfully
  • cancelled - Order was cancelled

View Order Details

GET /api/orders/:id — Auth: JWT
Get detailed information about a specific order

üí° Tip:

Use Order IDs from "View My Orders" to get detailed information about specific orders

Cancel Order

POST /api/orders/:id/cancel — Auth: JWT
Cancel an existing order (only if not completed)

‚ö†Ô∏è Important Notes:

  • Orders can only be cancelled if they are not completed or already cancelled
  • Once an order is confirmed or preparing, it cannot be cancelled
  • Use Order IDs from "View My Orders" to cancel specific orders
  • This action cannot be undone

Rate Order

POST /api/orders/:id/rate — Auth: JWT
Rate and review a completed order

üí° Rating Guidelines:

  • You can only rate completed orders
  • Be honest and constructive in your review
  • Your feedback helps improve our service
  • Include details about food quality, service, and delivery
