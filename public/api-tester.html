Cafe Backend APIs

Fast & Reliable backend API

Register

POST /api/auth/register — Auth: None
Register a new user (customer, admin, etc.)

Login

POST /api/auth/login — Auth: None
Login and receive JWT token

Get Me

GET /api/auth/me — Auth: JWT
Get current user profile

Get Orders (Admin Only)

GET /api/orders — Auth: Admin JWT
Get all orders (admin only)

Get My Orders

GET /api/orders/user — Auth: JWT
Get orders for the logged-in user

Create Order

POST /api/orders — Auth: JWT
Create a new order

Get Order by ID

GET /api/orders/:id — Auth: JWT
Get order details by order ID
